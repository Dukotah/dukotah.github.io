# Copper Bay Labs — hub

The public front door for the Copper Bay Labs tool family, served at the user root
**https://dukotah.github.io/** (repo `Dukotah/dukotah.github.io`). Static, zero-secret, free on
GitHub Pages.

- Lists every live tool, grouped (ship-safety suite · business tools) — internal-link SEO that
  funnels link equity to the subpath tools.
- `sitemap.xml` lists the root + all live tool URLs (same origin) to aid discovery.
- When a new tool ships, add a `.tool` card here and an entry to `sitemap.xml`, then push
  (Pages rebuilds on push to main).

Distinct from `overseer-dashboard/` (internal status board at /overseer-dashboard/).
